About Cloud Collaborator

The main motivation behind Cloud Collaborator was to have a real time editing and coding experience , in which multiple teammates can start collaborating on code at the same time sitting at different locations. The idea behind Cloud Collaborator was also driven behind the aim of making a coding ready environment available to anyone having access to the internet and that too anywhere they are .
